# CEDCO R02 Agent Flow

CEDCO R02 Recepcion y Agendamiento flow:

1. Saludo profesional en español Colombia.
2. Deteccion de intencion: informacion, agenda, sede, convenio, humano, urgencia o desconocida.
3. Respuesta desde conocimiento aprobado cuando aplique.
4. Consulta de disponibilidad antes de ofrecer horarios.
5. Creacion de cita interna si el usuario acepta.
6. Confirmacion de cita con datos no sensibles.
7. Fallback cuando el conocimiento o disponibilidad no resuelve.
8. Handoff a humano/PBX si hay urgencia, baja confianza, solicitud humana o error operativo.
9. Cierre claro y breve.
