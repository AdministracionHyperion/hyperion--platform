# CEDCO D02 Handoff And Compliance

## Handoff

CEDCO D02 recomienda handoff cuando:

- El usuario pide humano.
- Hay urgencia.
- La confianza es baja.
- Hay riesgo de politica.
- El usuario insiste sobre convenio o servicio desconocido.
- Una integracion real seria necesaria pero no existe.
- El caso queda fuera de alcance.
- Hay opt-out y se requiere cierre controlado.

## Urgencia

La urgencia deriva a humano o linea correspondiente sin diagnosticar y sin triage clinico. El agente
no interpreta sintomas ni toma decisiones clinicas.

## Opt-out

Opt-out se respeta y se marca como cierre/handoff. No se insiste con llamadas ni se guarda dato
personal real en el dominio.

## No diagnostico

CEDCO D02 bloquea diagnostico medico, recomendaciones medicas e interpretaciones clinicas.

## No triage clinico

CEDCO D02 bloquea triage, decisiones clinicas, lectura de resultados clinicos e historia clinica.

## Habeas data

El dominio usa referencias seguras, metadata sanitizada y auditabilidad. No guarda telefonos reales,
documentos reales, historia clinica real, transcript crudo ni audio crudo.
