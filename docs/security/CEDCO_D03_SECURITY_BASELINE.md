# CEDCO D03 Security Baseline

## Prohibiciones

- No datos reales.
- No Excel real.
- No facturas reales.
- No fotos reales.
- No seriales reales.
- No responsables reales.
- No proveedores reales.
- No valores contables reales.
- No import/export hasta loop especifico.

## Requisitos Futuros

- Todo tenant-scoped.
- Audit log obligatorio.
- Roles de aplicacion futuros.
- Sanitizacion de metadata.
- Revisiones antes de DB/API.
- Import/export bloqueado por policy gates hasta aprobacion explicita.
